E-Commerce Product Dashboard

A modern, responsive e-commerce product dashboard built with React, Next.js, TypeScript, and Tailwind CSS. Features real-time search, filtering, sorting, cart management, and dark mode.

 - Product data provided by [DummyJSON](https://dummyjson.com/)
 - Icons from Heroicons (via Tailwind)
 - Built with Next.js and React
